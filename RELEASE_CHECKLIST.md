## 2. 浏览器测试流程

### 2.1 启动服务
```bash
node dist/cli.js web
```
- [ ] 服务正常启动，无错误日志
- [ ] 端口 8443 正常监听

### 2.2 登录页面测试
打开 https://localhost:8443

- [ ] 页面正常加载，无白屏
- [ ] 登录表单显示正常
- [ ] 可以成功登录

### 2.3 主界面测试
- [ ] 会话列表正常显示
- [ ] 侧边栏菜单正常展开/收起
- [ ] 顶部工具栏按钮正常响应

### 2.4 新建会话测试
- [ ] 点击「新对话」按钮正常
- [ ] 模式选择器（full-access/default/auto-edit/native）正常

### 2.5 Chat 模式测试
- [ ] 切换到「对话返回」视图
- [ ] 消息自下而上显示
- [ ] 消息无闪烁/抖动
- [ ] 代码块渲染正常
- [ ] 代码复制按钮正常

### 2.6 终端模式测试
- [ ] 切换到「原生返回」视图
- [ ] 终端显示正常
- [ ] 终端可以正常交互

### 2.7 输入框测试
- [ ] 输入框可以正常输入
- [ ] Enter 发送消息正常
- [ ] Shift+Enter 换行正常
- [ ] Cmd+C/Ctrl+C 复制选中文本正常
- [ ] Cmd+V/Ctrl+V 粘贴正常
- [ ] Cmd+A/Ctrl+A 全选正常

### 2.8 悬浮窗快捷键测试
- [ ] 点击 ⌨ 按钮打开悬浮窗
- [ ] Ctrl+C 按钮发送中断信号

- [ ] Ctrl+L 按钮清屏
- [ ] 上下方向键导航正常
- [ ] Enter 按钮发送正常

### 2.9 会话管理测试
- [ ] 会话列表显示正常
- [ ] 切换会话正常

- [ ] Resume 按钮显示（如果有 claudeSessionId）
- [ ] Resume 功能正常

### 2.10 草稿持久化测试
- [ ] 输入框内容刷新后保留

### 2.11 WebSocket 断线重连测试
- [ ] 刷新页面后自动重连
- [ ] 断线期间消息不丢失
