# wand-cli 用户体验测试邀请

## 产品简介

**wand-cli** 是一个基于 Web 的 CLI 工具操作台，让你可以在浏览器中操作本地 CLI 工具（如 Claude Code）。

### 核心功能
- 🌐 浏览器访问本地 CLI 工具
- 💬 Chat 模式：Markdown 渲染、代码高亮
- 📁 文件浏览器：查看项目文件结构
- 🔧 终端模式：原生 PTY 交互体验
- ⌨️ 快捷键支持：Ctrl+C, Ctrl+D, Ctrl+L 等

---

## 测试环境

**访问地址：** https://localhost:8443

**测试账号：** 请使用配置文件中设置的密码登录

## 测试任务清单

请完成以下测试任务，并记录遇到的问题和建议：

### 任务 1：登录流程
- [ ] 打开 https://localhost:8443
- [ ] 输入密码登录
- [ ] 记录：登录页面显示是否正常？有无错误提示？

### 任务 2：新建会话
- [ ] 点击「新对话」按钮
- [ ] 选择模式（full-access / default / auto-edit / native）
- [ ] 记录：模式选择器是否清晰易懂？

### 任务 3：Chat 模式体验
- [ ] 切换到「对话返回」视图
- [ ] 输入一个问题并发送
- [ ] 观察消息显示方向（应自下而上）
- [ ] 检查代码块渲染和复制按钮
- [ ] 记录：消息有无闪烁？代码高亮是否正常？

### 任务 4：终端模式体验
- [ ] 切换到「原生返回」视图
- [ ] 观察终端显示
- [ ] 尝试输入命令
- [ ] 记录：终端交互是否流畅？

### 任务 5：输入体验
- [ ] Enter 键发送消息
- [ ] Shift+Enter 换行
- [ ] Ctrl+C 复制选中文本
- [ ] Ctrl+V 粘贴
- [ ] 记录：快捷键是否正常工作？

### 任务 6：悬浮快捷键
- [ ] 点击 ⌨ 按钮打开悬浮窗
- [ ] 尝试 Ctrl+C、Ctrl+D、Ctrl+L 按钮
- [ ] 记录：悬浮窗是否易用？

### 任务 7：文件浏览器
- [ ] 点击侧边栏 Files 标签
- [ ] 展开目录查看文件
- [ ] 记录：文件树加载是否正常？

### 任务 8：会话管理
- [ ] 查看侧边栏 Sessions 标签
- [ ] 切换历史会话
- [ ] 刷新页面，检查草稿是否保留
- [ ] 记录：会话切换是否流畅？
