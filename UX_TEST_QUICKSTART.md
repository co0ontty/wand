## 快速测试清单（30 分钟版）

如果你时间有限，请完成以下核心测试：

### □ 1. 登录（2 分钟）
- [ ] 打开页面，看到登录表单
- [ ] 输入密码登录成功

### □ 2. 新建会话（5 分钟）
- [ ] 点击「新对话」按钮
- [ ] 选择「Claude」预设
- [ ] 选择模式（推荐 default）
- [ ] 会话正常启动

### □ 3. 聊天模式测试（10 分钟）
- [ ] 切换到「对话返回」视图
- [ ] 输入消息并发送（Enter 键）
- [ ] 消息正常显示（自下而上）
- [ ] 代码块渲染正常
- [ ] 可以复制代码
- [ ] Shift+Enter 可以换行

### □ 4. 终端模式测试（5 分钟）
- [ ] 切换到「原生返回」视图
- [ ] 终端显示正常
- [ ] 可以正常输入和交互

### □ 5. 文件浏览器（5 分钟）
- [ ] 点击侧边栏「Files」标签
- [ ] 可以展开目录
- [ ] 文件列表正常显示

### □ 6. 快捷键测试（3 分钟）
- [ ] 点击 ⌨ 打开悬浮窗
- [ ] 尝试 Ctrl+C 按钮
- [ ] 尝试 Ctrl+D 按钮
